# UI Test with TestNG And Selenium in Java

---

This project was created to practice software testing using **Java** and **Selenium**.  
It includes functional tests for the **SauceDemo** website, covering pages such as **Login** (valid and invalid), **Cart**, and **Payment**.

---

### Technologies Used
- **Language:** Java
- **Automation Tool:** Selenium WebDriver
- **Test Framework:** TestNG
- **Design Pattern:** Page Object Model (POM)
- **Build Tool:** Maven
- **Browser:** Chrome
